函数式接口

interface Predicate:test()

断言性接口：Predicate 主要用到test方法 其中参数t为输入参数，如果输入参数符合断言则返回true，否则false

interface Consumer:accept()

消费性接口：Consumer 只接收一个参数t，但是没有返回值。

interface Function<T, R>:apply()

功能性接口：Function 接收一个功能参数t，并返回一个功能结果R。

interface Supplier:get()

供给性接口：Supplier 不接收任何参数 但有返回值
