Maven（==约定大于配置==）

用途：

  1. 再写Java程序的时候，有的时候要导入大量的jar包，我们要导入
  2. 我们想要一个东西来帮我们自动导包

一、环境变量的配置

  1. 配置M2_HOME： 安装maven的bin目录下
  2. 配置MAVEN_HOME：安装maven的目录
  3. 在path下面配置MAVEN_HOME的bin目录；就是M2_HOME的位置
