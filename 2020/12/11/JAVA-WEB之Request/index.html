关于乱码问题：

1.首先是Tomcat 的问题，如果在界面加载控制台出现乱码，就改logging.properties（java.util.logging.ConsoleHandler.encoding = GBK）最后一个有关编码的UTF-8改成GBK;

2.如果是接收的时候出现数据的乱码，就用request.setCharacterEncoding (“utf-8”);就可以了
