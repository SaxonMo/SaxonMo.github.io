Python

一、与Java不同的地方（语法）

一、格式化输出和强制类型转换

**格式化输出：%d %(field)**前面是类型后面是属性名字

**强制类型转换：a=int(a)**需要何种类型就用类型加括号就可以转换了

占位输出，空一格输出就要给一个

二、算数运算符

  • x**y，x的y次方

  • x//y,x➗y的值，整除，不会有小数点和四舍五入；要有小数点的话，需要使用x/y
